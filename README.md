 **Programming Errors in C**
 
There are mainly five types of errors exist in C programming:

- Syntax error
- Run-time error
- Linker error
- Logical error
- Semantic error
 
## Example code for Syntax Error

```C
#include <stdio.h>  
int main()  
{  
    a = 10;  
    printf("The value of a is : %d", a);  
   return 0;  
}  
```
